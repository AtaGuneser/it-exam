# Patient Management App

Basit bir hasta yönetim sistemi. Bu repo iki bölüm içerir: **backend** (API) ve **frontend** (React).  

---

## Teknolojiler

**Backend**
- Node.js, TypeScript
- Express
- MongoDB (native driver)
- Zod (validation)
- JWT authentication

**Frontend**
- React 18+
- Vite
- TanStack Router
- TanStack Query
- TanStack Table
- Shadcn UI (veya alternatif UI)
- React Hook Form + Zod
